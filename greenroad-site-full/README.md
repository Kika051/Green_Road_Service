# Green Road Services Frontend

## Lancer le projet

1. Ouvre un terminal dans ce dossier.
2. Installe les dépendances :
```
npm install
```
3. Lance le projet :
```
npm start
```

## Pages disponibles :
- `/` : Accueil
- `/booking` : Réserver une course
- `/login` : Connexion client
- `/admin` : Tableau de bord admin
